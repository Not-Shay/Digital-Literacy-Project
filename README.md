# Digital Literacy Project

**Name:** Lakshay Singh  
**Course:** CSE0001 – Digital Literacy  
**University:** VIT Bhopal University  

---

## Project Overview

This project is a Digital Literacy Portfolio created as part of the CSE0001 course. It demonstrates understanding of essential digital skills, including creating visual content, building a professional online presence, using coding and collaboration platforms, practicing email etiquette, and understanding cybercrime awareness.

The project is organized into five tasks, each focusing on a different aspect of digital literacy.

---


---

## Task 1 – Digital Literacy Infographic

Created a visual infographic using Canva covering:
- Digital literacy basics  
- Safe internet practices  
- Useful digital tools  

---

## Task 2 – Digital Portfolio

Created professional profiles on:
- GitHub  
- LinkedIn  
- Kaggle  

These platforms will be used to showcase projects, connect with professionals, and build technical skills over time.

---

## Task 3 – Platforms

- Solved a beginner-level coding problem on HackerRank  
- Created a Google Form quiz on digital literacy  
- Linked responses to Google Sheets  

---

## Task 4 – Email Etiquette

- Drafted two professional emails  
- Created a social media do’s and don’ts checklist  
- Learned proper communication practices  

---

## Task 5 – Cybercrime Awareness

- Created a case study on UPI fraud  
- Developed a prevention checklist  
- Learned safe online practices and reporting methods
